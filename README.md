# esp32-cam-to-led-matrix

simple example sketch to bring the esp32-s3 cam board video stream on a small ws2812 matrix.

## HW

[amazon diymore - ESP32 S3 CAM](https://www.amazon.de/gp/product/B0F4D8ZY6L/)

> ESP32-S3-WROOM CAM
> ESP32-S3-N16R8
> 2640 Camera
> TF Card Module
